Writeup Disk Forensic

Gemastik XII Final

Posted by rmn0x01 on Wednesday, October 30, 2019

Soal forensik pembuka final Gemastik XII di Telkom University. Poin 200

Intro

File yang diberikan berupa

$ file forensic.img 
forensic.img: data

TLDR

  1. Perbaiki file
  2. Mount file
  3. Get the flag

Full Steps

$ mount -o loop forensic.img mont/
